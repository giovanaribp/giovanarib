## . ݁ SelfCoded Portfolio . ݁.

#### 💻 | About the project:
This website was my first major project as a **self-taught** web developer.

I built it from scratch, starting with the design in Figma, visualizing how I wanted the elements to open and focusing mainly on a clean aesthetic, to finishing the code in VS Code. My main goal was for the website to work seamlessly, opening different sections without reloading the page, and this was possible by using pop-ups and sections.
<hr />

#### 🌐 | Languages:
The website was built using HTML, CSS and JavaScript, which are the languages I currently have the most experience with. I also used free support scripts, such as Lightbox2 to expand the existing images into a larger view, and GSAP for some page animations. The icons were sourced from the free website Flaticon.
<hr />

#### 🖇️ | Learning:
When I finally finished my first major project, I realized that programming is (and always will be) a long learning process, where there is no such thing as too much research. Small bugs would pop up that took me hours to figure out, only to turn out to be something simple in the end, but those are now permanently etched in my mind!
